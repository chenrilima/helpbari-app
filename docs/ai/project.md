# HelpBari

HelpBari é um aplicativo Flutter destinado ao acompanhamento de pacientes bariátricos.

O projeto utiliza:

- Flutter
- Riverpod
- GoRouter
- Clean Architecture
- Feature First
- Design System próprio

Objetivos:

- acompanhamento de peso
- hidratação
- vitaminas
- consultas
- exames
- alimentação
- notificações
- integração com Supabase

Toda nova feature deve seguir exatamente o mesmo padrão arquitetural.
